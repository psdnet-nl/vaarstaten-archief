# Vaarstaten archief PSDnet.nl

## Description
A web app that visualizes data, 'vaarstaten' in Dutch, about the ferry services in Zeeland, the Netherlands.

## The Data
The data used in this project is gathered from the Zeeuws Archief archive in Middelburg, the Netherlands: 123 Provinciale Stoombootdiensten in Zeeland (PSD), 3.2.6 (917-920).

## Usage
Run the python script to get the web app running, don't forget to look at requirements.txt if your manually running the app.
